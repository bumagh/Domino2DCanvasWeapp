# 多米诺游戏 canvas 开发

示例相关说明查阅[新手教程](https://developers.weixin.qq.com/minigame/dev/guide/develop/start.html)

## 源码目录介绍

## 问题

1. 开始游戏流程繁琐，需要让玩家点击屏幕开始游戏
2. 游戏过程中没有音效
3. 游戏结束没有分享功能
4. 没有皮肤道具系统
5. 没有排行榜功能
6. 没有存档功能
7. 积分奖励不合理

## 解决方案

1. 优化开始游戏流程，简化玩家操作
2. 添加音效，提升游戏体验
3. 增加游戏结束分享功能，提升用户互动
4. 引入皮肤道具系统，增加游戏趣味性
5. 实现排行榜功能，增强竞争性
6. 添加存档功能，方便玩家继续游戏
